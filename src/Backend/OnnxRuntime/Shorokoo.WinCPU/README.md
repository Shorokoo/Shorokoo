# Shorokoo.WinCPU

[Shorokoo](https://github.com/Shorokoo/Shorokoo) execution backend for
**Windows x64 (CPU)**, powered by ONNX Runtime.

```bash
dotnet add package Shorokoo
dotnet add package Shorokoo.WinCPU
```

Reference exactly one backend package per application. Shorokoo discovers the
backend at first use — no configuration needed.

Documentation: https://github.com/Shorokoo/Shorokoo
